User Interaction Data Analysis

An in-depth analysis of website user interaction data with insights & suggestions

Scroll

Glance:

Information

This is a website user interaction data analytics project assigned to me by a tech company managing an anime/manga related website as part of an evaluation process. I got to learn about key metrics associated with user interaction data like page views, bounce rates, average time spent etc.


Below are the 3 tasks I was assigned, each based on 3 different articles/page present in the company website:


🔹Review the article “𝘢𝘳𝘵𝘪𝘤𝘭𝘦 𝘯𝘢𝘮𝘦 𝘳𝘦𝘥𝘢𝘤𝘵𝘦𝘥 𝘥𝘶𝘦 𝘵𝘰 𝘱𝘰𝘴𝘴𝘪𝘣𝘭𝘦 𝘳𝘪𝘨𝘩𝘵𝘴 𝘪𝘯𝘧𝘳𝘪𝘯𝘨𝘦𝘮𝘦𝘯𝘵” and imagine it has user data such as page views, average time spent, and bounce rate. Analyze these metrics and create visualizations (bar graphs, pie charts) to identify engagement trends.

Suggest two strategies to increase the average time spent on the page based on your analysis.


🔹Analyze user interaction data for “𝘢𝘳𝘵𝘪𝘤𝘭𝘦 𝘯𝘢𝘮𝘦 𝘳𝘦𝘥𝘢𝘤𝘵𝘦𝘥 𝘥𝘶𝘦 𝘵𝘰 𝘱𝘰𝘴𝘴𝘪𝘣𝘭𝘦 𝘳𝘪𝘨𝘩𝘵𝘴 𝘪𝘯𝘧𝘳𝘪𝘯𝘨𝘦𝘮𝘦𝘯𝘵” and propose an A/B testing strategy.

Recommend content changes to test (e.g. different headlines, visuals) that may improve user retention and reduce bounce rate.


🔹Use “𝘢𝘳𝘵𝘪𝘤𝘭𝘦 𝘯𝘢𝘮𝘦 𝘳𝘦𝘥𝘢𝘤𝘵𝘦𝘥 𝘥𝘶𝘦 𝘵𝘰 𝘱𝘰𝘴𝘴𝘪𝘣𝘭𝘦 𝘳𝘪𝘨𝘩𝘵𝘴 𝘪𝘯𝘧𝘳𝘪𝘯𝘨𝘦𝘮𝘦𝘯𝘵” as a case study.

Develop a report that outlines user segmentation based on demographics or behavior (e.g., age group, returning vs. new visitors) and suggest ways to tailor content for these segments.


What I Helped Achieve

🔹Analysed trends in user behavior across several months, identifying key insights on engagement and retention.

🔹Visualised key metrics using bar graphs and line charts to compare bounce rates and time spent on pages.

🔹Suggested actionable strategies to increase user interaction, including content optimisation and A/B testing based on detailed user segmentation.

🔹Developed recommendations for improving engagement through tailored content based on user demographics and behavior.


This project helped me test my skills in data analytics, visualisation, and storytelling, and I am looking forward to applying these knowledge in my future works!

Tools Used: Power BI, Excel, Canva, SimilarWeb
