Modulus Function in Mathematics

Modulus function (as absolute value - I like to think of it as a jail for the element i.e. whatever goes in comes out clean neither positive nor negative) in Mathematics is used to find the magnitude of a number, represented as |x|, read as 'mod of x'. This produces a non-negative value, regardless of the input.

You can think of it as the distance of that number from zero on a number line, without considering the direction in which the number lies.

The equivalent in spreadsheets of the 'modulus as absolute value' that is used in mathematics is the absolute function (=abs(number)) which will return the non-negative value of a number regardless of its sign.
