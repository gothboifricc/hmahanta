Spotify Music Data Analysis Dashboard

A dashboard for spotify music data covering key metrics like popularity by artists, genre etc.


September 15, 2024
Scroll

Glance:

Information

I had the chance to explore a Spotify dataset and uncover some fascinating insights into music trends and popularity.

The project involved data cleaning, transformation, creating elements and visualisation using Excel, Canva, Photoshop, and Power BI. This was more of a dashboard designing project than an in-depth analysis as it allows the user interacting with the dashboard to specify criterion to compare metrics across the dataset.
